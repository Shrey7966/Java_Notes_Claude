- Literal

Any constant value which can be assigned to the variable is called literals.

int x = 10 ;

Ex:
    int is datatype or keyword
    x is variable or identifier
    10 here is constant value / literal.

- Integral Datatypes

for integral data types (byte, short, int, long ) we can specify literal form in following ways

int x = 10; decimal form (0 to 9)
int x = 010; octal form (0 to 7)
int x = 0X10; base_16 (0to 9, a to f)


Ex:

int x = 10; valid
int x = 0786 invalid CE: integer number too latge
int x = 0777; valid
int x = 0XFace; valid
int x = 0XBeer; invalid as it should be valid from a to f
